## Frontend (Next.js)

Интерфейс для просмотра каталога автомобилей из БД.

### Основные возможности

- Авторизация по логину/паролю `admin` / `admin123` (через backend JWT).
- Страница списка `/cars`:
  - адаптивная сетка карточек (ПК / планшет / мобильный),
  - сортировка (новые/старые, цена, пробег),
  - фильтры: марка, год от/до, цена до, пробег до,
  - пагинация.
- Страница детализации `/cars/[id]`:
  - фото, характеристики, цена,
  - описание автомобиля,
  - ссылка на исходное объявление на CarSensor.
- Переключение на переведённый текст:
  - если есть переводы, заголовок, дилер, регион и описание показываются на **русском**, затем на **английском**, и только потом оригинал на японском.

### Запуск в dev-режиме

```bash
cd frontend
cp .env.example .env.local
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:3000`.

Переменные окружения:

- `NEXT_PUBLIC_API_URL` — URL backend API (по умолчанию `http://localhost:4000`).
